# DrawLab
Technical vector drawing app for OSX

- [ ] Import SVG 
- [ ] Export SVG
- [ ] Export CSS3 with depth and reusable Gradients/Shadows (clipboard)
- [ ] Import CSS3 
- [ ] Gesture zoom
- [ ] Zoomable Rulers
- [ ] Live update (Target SVG to Target CSS3 file)
